## 🔥 功能介绍

### ✨ 支持多种手势识别

- 打开手掌 / 握拳
- 竖起食指 / 比耶
- 大拇指向上 / 向下

### ✨ 支持多种操作控制

- 绑定任意软件
- 绑定任意快捷键
- 模拟鼠标动作（ 滚轮 / 光标 / 点击 ）
- 模拟特定行为（ 调节音量 / 切换歌曲 ）

## 🧙🏻 快速上手

### 🔮 开箱即用

直接在 [release](https://github.com/RylanBot/awesome-hands-control/releases) 中下载已经打包好的安装包

### 🔮 二次开发

如果你熟悉 Web 前端技术且对源码感兴趣，可以根据以下命令，在本地启动这个程序

```sh
npm install
npm run dev
```
